SortingBenchmark.py
-------------------

A benchmarking application to compare the performance of different sorting algorithms.

How to Run:
  python SortingBenchmark.py

Requirements:
  - Python 3.x
  - No external packages required

Usage:
  1. Run the script
  2. Follow the on-screen prompts
  3. View benchmark results comparing algorithm performance


PURPOSE
-------
This lab work focuses on:
  - Measuring and comparing execution times of sorting algorithms
  - Understanding time complexity through practical benchmarks
  - Analyzing algorithm performance with different dataset sizes

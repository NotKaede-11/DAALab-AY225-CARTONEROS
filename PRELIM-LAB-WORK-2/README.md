# Sorting Algorithms Visualizer (GUI Application)

A full-featured GUI sorting application built with Tkinter.

## Files

- `Alg.Sorter.py`: The main GUI application script.
- `dataset.txt`: Sample dataset containing integers.

## Features

- **Graphical User Interface:** Built using Python's `tkinter`.
- **Three Sorting Algorithms:**
  - Bubble Sort
  - Insertion Sort
  - Merge Sort
- **Controls:** Features options to select algorithms and sort orders (Ascending/Descending).

## How to Run

1. Ensure you have Python 3.x installed.
2. Run the application:
   ```bash
   python Alg.Sorter.py
   ```

## Requirements

- Python 3.x
- `tkinter` (included with standard Python installations)
